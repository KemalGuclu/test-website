Introducing HTML

The HyperText Markup Language— HTML, is the most fundamental building component of the Web, and it establishes the web content's meaning and structure. Other technologies are also used to enhance the webpage's appearance or presentation (CSS) or functionality and behavior (JavaScript).

Professional text editors can help you create and edit web pages. However, it is recommended for beginners to learn HTML using a simple text editor such as Notepad or TextEdit because it is simpler and easier to understand.

What is CSS?

CSS stands for Cascading Style Sheets and is a way of styling and presenting HTML. The style sheet is responsible for the page's appearance, whereas the HTML is responsible for the page's meaning or content.

CSS can be used to define things like:

  • Fonts, sizes and colors
  • Position, alignment and distribution of items
  • Margins, paddings, height, width and borders
  • Visibility, opacity, text effects and transformations
  • Animations
  • Etc.
